项目示例

1.个人网站

技能：生成式 AI 工具与 GitHub

使用 HTML 模板和生成式 AI 工具搭建了这个个人网站。 在编码、排查问题以及页面结构与设计优化过程中使用 AI， 体现了我将 AI 融入工作流程、提升效率与执行速度的能力。

2.大规模媒体数据与市场行为分析

技能：BigQuery、SQL、Python 与 Stata

基于 BigQuery 调取和处理来自 GDELT 的大规模结构化媒体数据， 并与投资交易数据进行整合分析。基于媒体情绪指标构建分析变量， 研究外部信息信号与市场行为、风险信号以及投资者反应变化之间的关系。

3.社交媒体情绪与中国股票市场预测

技能：网页爬取与 R 语言机器学习

通过网页爬取收集投资者情绪文本和股票市场数据，完成预处理和特征工程， 并使用中文 NLP、ANN、SVM 和遗传算法模型预测股价走势。 同时结合 MACD、RSI 和动量等金融指标进行分析。

4.美国个人消费预测

技能：R 语言时间序列预测

在 R 中完成时间序列数据清洗与缺失值填补、季节性分解， 并比较 drift、指数平滑和 ARIMA 等预测方法来预测美国个人消费。 使用 MAE、MAPE 和 MASE 评估模型表现。

5.客户评论主题模型分析

技能：情感分析与 R 语言主题模型

对评论文本进行处理，构建文档词矩阵，并使用 LDA 主题模型和 coherence score 识别正面与负面评论中的关键主题。结果显示，员工、房间和位置是影响满意与不满意的重要因素。

6.ICO 募资结果预测

技能：R 语言机器学习

完成端到端数据预处理，并建立决策树、随机森林、朴素贝叶斯、KNN、SVM 和 ANN 等分类模型， 用于预测 ICO 项目是否能够达到募资目标。通过混淆矩阵、ROC 曲线和交叉验证对模型进行比较。
